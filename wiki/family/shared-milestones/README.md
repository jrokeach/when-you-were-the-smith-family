# Shared milestones

Family-level milestones: moves, a new sibling joining the family, an adoption, a change in household composition, a significant diagnosis that affected the whole family, a parent changing jobs in a way the kids felt.

## How to file

- One page per milestone: `YYYY-short-slug.md`.
- `subjects:` usually `[family]` plus every kid present at the time.
- If each kid processed the milestone differently, add per-kid `reflections/` pages.
