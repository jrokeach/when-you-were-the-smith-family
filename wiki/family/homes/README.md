# Homes

Places the family has lived. One page per home, most recent first. Pages include: address (or neighborhood — your call on specificity), dates, layout notes, what the kid(s) remember about it, distinctive features, why you left.

## How to file

- One page per home: `YYYY-descriptive-name.md` (e.g. `2024-maple-street.md`).
- `subjects: [family]` usually. Some homes may have a `subjects:` that includes a child if they've journaled about it.
