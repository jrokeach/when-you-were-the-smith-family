# Traditions

Family rituals — holiday ones, weekly ones, the small recurring shapes of your household's life. These drift and evolve; a KB that captures them at various points is more interesting than one that captures only the "current" version.

## How to file

- One page per tradition: `slug.md` (e.g. `sunday-pancakes.md`, `thanksgiving.md`).
- If a tradition meaningfully changed, a new page or a new dated section within the existing page, not an overwrite.
