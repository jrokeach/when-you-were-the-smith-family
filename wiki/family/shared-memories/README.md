# Shared memories

Discrete shared events that don't fit neatly into trips, traditions, or lore. A Saturday that became a memory. A walk you all took. The afternoon the power went out.

## How to file

- One page per memory: `YYYY-MM-short-slug.md` or `YYYY-short-slug.md`.
- `subjects:` lists all kids involved.
- If each kid had a distinct experience worth preserving, add per-kid `memories/` pages and link between them.
