# Shared firsts

Firsts for the family unit, not for any individual kid: first pet, first home, first trip abroad, first time you all slept in a tent, first snow in the new climate.

## How to file

- One page per first: `first-<thing>.md` or `YYYY-first-<thing>.md`.
- `subjects:` is usually `[family]` plus every kid who was present.
