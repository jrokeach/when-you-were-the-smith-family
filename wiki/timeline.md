# Timeline

Chronological view across all children and family-level content. Maintained by the agent — when a page with a `date:` field is added or updated, a timeline entry is reconciled here.

## Format

```
## YYYY

### YYYY-MM — month label (optional)

- YYYY-MM-DD — [short description](path/to/page.md) — subjects: [ava, noah]
```

Granularity is flexible — a single "2019" entry is fine if that's all you know; a specific date is better when available.
